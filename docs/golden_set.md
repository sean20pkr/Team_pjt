# Golden Set 100

기준:
- 목적: mock 데이터 기준으로 질문 표현이 달라져도 같은 의미를 안정적으로 잡는지 검증
- 구성: 같은 의미의 질문을 4개씩 바꿔 총 100문항으로 구성
- 판정: 각 문항의 정답 요지와 핵심 토큰이 일치하면 통과
- 보고문 계열은 `보고서 / 보고문 / 마감보고 / 마감초안 / 임원보고용`을 같은 의도로 묶어 평가

## 평가 방식
- 답변자 에이전트: 각 질문에 대해 실제 봇처럼 답변
- 채점자 에이전트: 근거, 정확, 에스컬레이터, 금액단정금지 4기준으로 O/X 채점
- 검토자 에이전트: 흔들리는 판정은 다시 확인
