# Forecast Report Golden Set

기준:
- 목적: `전망·보고 지원` 품질을 mock 데이터 기준으로 검증
- 검증 방식: 답변자 / 채점자 멀티에이전트
- 채점 항목: `근거`, `정확`, `에스컬레이터`, `금액단정금지`
- 판정: 각 항목 `O/X`, 종합 `O/X`

## 8문항 세트

| ID | 유형 | 질문 | 기대 답변 요지 | 판정 기준 |
|---|---|---|---|---|
| F1 | 전망 | 2026년 6월 전사 월초 전망의 중심값과 범위를 제시해줘. | 보고형 문장으로 중심값 260 내외, 범위 255~265 추정 | 범위/추정 표현 |
| F2 | 방향 | 2026년 6월 전사 월초 전망이 전월 대비 어느 방향인지 한 줄로 말해줘. | 보합~소폭 상방 | 과도한 단정 금지 |
| F3 | 채널 | 2026년 6월 채널별 방향성을 FC, GA, GFC 중심으로 요약해줘. | FC 보합~소폭 상방, GA 보합, GFC 보합~약상방 | 채널별 방향성 일치 |
| F4 | 영향요인 | 2026년 6월 전사 전망에 가장 중요한 영향요인 3개를 우선순위로 말해줘. | FC 방어, 건강 경쟁 압박, 신채널 전환 속도 | 우선순위+근거 |
| F5 | 조건 | 2026년 6월 전사 전망이 상단/하단으로 흔들릴 수 있는 조건을 2개만 말해줘. | 상단: 신채널 전환 개선 / 하단: 건강 경쟁 재강화 | 조건형 표현 |
| F6 | 경쟁비교 | 2026년 6월 경쟁보험사 비교 관점에서 지금 보고에 넣을 한 줄을 써줘. | 경쟁 압박은 남아 있으나 전환 회복 시 방어 여지 | 비교 관점 포함 |
| F7 | 보고문 | 임원 보고용으로 바로 쓸 수 있는 1문장 초안을 써줘. | 259억 수준의 현재 현황, 전월/전년 비교, 다음달 범위, FC/건강/신채널/Special_Product 실행전략을 압축한 1문장 | 보고 초안 형태 |
| F8 | 확정성 | 2026년 6월 전사 월초를 숫자로 확정할 수 있어? | 확정은 불가하지만 추정 중심값 261.8 내외, 범위 259.0~264.6, 추정 논리까지 제시 | 금액단정 금지 + 숫자 추정 + 근거 |

## 자체 검증 결과

- 답변자/채점자 멀티에이전트 검증 결과: **8문항 모두 O**
- 미래 금액은 숫자 단정 없이 `추정` 또는 `확정 불가`로 처리하되, 가능한 경우 중심값과 범위, 추정 논리, 실행전략까지 함께 제시
- 현재 시점에서는 `전망·보고`의 중심값/범위/조건형 문장 품질이 기준을 충족함

## 표현 차이 기준

| 표현 | 응답 성격 | 포함 범위 | 길이 |
|---|---|---|---|
| 보고문 | 한 문단 압축형 | 현재 현황 + 전월/전년 비교 + 다음달 범위 + 실행전략 핵심 | 짧음 |
| 마감초안 | 섹션형 보고서형 | 현재 현황 + 물량 + 영업효율 + 다음달 전망 + 추진전략 | 중간 |
| 영업현황 | 조회형 현황 요약 | 현재 월 실적 + 채널 순위 + Special_Product 분해 | 가장 짧음 |

## 운영 메모

- 이 문서는 `전망·보고` 정교화 작업의 기준 문서로 사용한다
- 추후 실제 BizRouter 연동 이후에도 같은 질문 세트로 재검증한다
